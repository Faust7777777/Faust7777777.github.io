# 区块链学信学生创业融资平台 V7.1

这个仓库，是我在 `2025 年 10 月 5 日` 做出来的一个比赛网页。

那时候我刚结束大学军训，对编程和代码几乎没什么概念。说得直接一点，就是不懂。可比赛要继续，团队后面需要有人把网站写出来，也需要有人把网站弄到公网能访问，这事最后就落到了我头上。

所以这个项目的来路，其实并不神秘。不是学会了才开始做，而是先被推上去，再边做边学。

## 当时我是怎么做的

写网站的时候，我没有老师带，也没有系统学过前端。很多东西都是硬着头皮试出来的：问 chatbot，复制，粘贴，改一改，再看能不能跑。现在回头给它起名，倒也可以叫一种非常原始的 vibe coding。

至于上线，更原始。

我当时查论坛，翻知乎，最后才摸到 GitHub Pages 这条路。那时候我甚至不会 `git` 命令，改网页很多时候靠的不是提交和回滚，而是删掉、替换、重新上传。今天看当然很笨，甚至有点好笑；但那时候确实就是这么一步一步弄上去的。

所以这个仓库里留下来的，不只是一个静态页面，也是一整套很生涩、很原始、但确实有效的摸索过程。

## 现在再看这个仓库

现在我的生活和工作已经离不开 Codex 和 Claude Code，也多少正规学过一点编程，像 Java 这样的语言，也算懂了一些。后来网站也做得比当时像样，至少知道什么叫版本管理，什么叫部署，什么叫把代码当工程来对待。

也正因为如此，我现在再看这个仓库，反而不会只觉得它粗糙。

它当然有很多问题。代码不规范，写法也稚嫩，里面甚至保留着一种很重的“能跑就先跑”的气味。但如果没有这一版东西，后面的很多事未必会有。它算不上什么像样的起点，却的确是一个起点。

我一直觉得，人后来能学会很多更正规的东西，但第一次靠自己把一个东西从本地弄到公网，那种感受不会有很多次。它很笨，也很真。

## 现在的处理

这个仓库目前被保留为一份早期作品存档。

我删掉了一些无关文件、外部统计脚本和不适合继续公开放着的附件，但页面主体还保留着当年的样子，因为我不想把这段痕迹修得太干净。

如果你今天点开它，觉得它有些生涩，有些用力过猛，甚至有点拧巴，那大概就对了。  
它本来就是我还什么都不太会的时候，自己一点点折腾出来的。
